# 業務アプリ 使い方ガイド

## アプリを開く

**パソコン:** デスクトップの「業務アプリ」アイコンをダブルクリック
**スマホ:** ホーム画面の「業務アプリ」をタップ

---

## ログイン

1. メールアドレスとパスワードを入力
2. 「ログイン」ボタンを押す

```
メールアドレス: （管理者から教えてもらったもの）
パスワード:     （管理者から教えてもらったもの）
```

**パスワードを忘れた場合:** 管理者に連絡してください

---

## ホーム画面

ログインすると、使えるアプリのボタンが並びます。

| ボタン | やること |
|--------|---------|
| 📝 作業入力 | その日の作業を記録する |
| 🔍 点検入力 | 設備・車両の点検結果を記録する |
| 📊 ダッシュボード | 実績をグラフで確認する（管理者・事務） |
| 🤖 事例生成 | 改善提案を作成する（管理者・事務） |

---

## 作業入力のやり方

1. 「📝 作業入力」を押す
2. 日付・担当者・作業内容を選ぶ
3. 「保存」ボタンを押す

**1分もあれば入力できます。**

---

## 点検入力のやり方

1. 「🔍 点検入力」を押す
2. 点検対象・担当者を選ぶ
3. 各項目に OK / NG / NA を押す
4. 「提出」ボタンを押す

---

## デスクトップアイコンの追加（Windows）

1. Chromeでアプリを開く
2. アドレスバーの右端に出る「⊕インストール」ボタンを押す
3. 「インストール」を選ぶ
4. デスクトップとスタートメニューにアイコンが追加される

---

## スマホのホーム画面追加（iPhone）

1. Safariでアプリを開く
2. 下の「共有」ボタン（□に↑）を押す
3. 「ホーム画面に追加」を選ぶ
4. 「追加」を押す

---

## 困ったときは

| 症状 | 対処 |
|------|------|
| ログインできない | パスワードが違う可能性あり → 管理者へ連絡 |
| 画面が白い | 画面を更新（F5キー）してみる |
| 「オフライン」と表示される | Wi-Fiまたはネットを確認して再接続 |
| アプリが遅い | ブラウザを再起動してみる |
