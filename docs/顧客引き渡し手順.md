# 顧客引き渡し手順

## 引き渡し前チェックリスト

- [ ] 本番URLでログイン確認
- [ ] 管理者アカウントをお客様専用のメールアドレスに変更済み
- [ ] デモ用パスワードを変更済み（seed で作成したデフォルトパスワードを削除）
- [ ] 各アプリが正常に開けること確認
- [ ] DBバックアップが動作することを確認
- [ ] お客様に「父親向け簡易マニュアル」を印刷して渡す

## 渡すもの

| 品目 | 形式 | 説明 |
|------|------|------|
| アプリURL | 口頭 + 紙 | ブックマーク済みのURL |
| 管理者メールアドレス | 紙 | ログイン用 |
| 管理者初期パスワード | 紙（封筒入り） | 初回ログイン後すぐ変更をお願いする |
| 父親向け簡易マニュアル | 印刷 | A4 1〜2枚 |
| 管理者向け運用手順 | 印刷 or PDF | ユーザー管理・ログ確認方法 |

## 引き渡し時の説明スクリプト

```
「このURLがお客様専用のアプリです。
ブックマークかデスクトップに保存してください。

Windowsの場合、Chromeで開くと右上に『インストール』ボタンが出ます。
押すとアイコンがデスクトップに追加されて、ふつうのアプリみたいに使えます。

スマホはホーム画面に追加できます（Safariの共有ボタン→ホーム画面に追加）。

管理者の方は、まず最初にパスワードを変更してください。
スタッフの追加・削除もここからできます。」
```

## 引き渡し後フォロー

- 1週間後: 使えているかメールで確認
- 1ヶ月後: 困っていることがないか確認
- 問い合わせ窓口を事前に伝えておく
