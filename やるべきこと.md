# AI Routing Rule v1

## ChatGPT
使う用途：
- 戦略設計
- ロードマップ
- 論点整理
- source化
- 記事構成
- 学習設計

使わない用途：
- 長い実装修正の反復
- IDE内でのコード変更の主作業

切替条件：
- 実装が3ファイル以上に跨ぐ → Claude Codeへ
- 公式情報の最新比較が必要 → ChatGPT web or Perplexity
- Google資料の読解・学習変換 → Gemini / NotebookLM

## Claude Code
使う用途：
- 実装
- リファクタ
- テスト
- コードレビュー
- 反復修正

使わない用途：
- 事業戦略の0→1整理
- ふわっとした比較検討

切替条件：
- 仕様が曖昧なら先にChatGPT
- 実装仕様が固定したらClaude Code

## Gemini
使う用途：
- NotebookLMで資料理解
- Nano Bananaで画像試作
- Google系資料との連携

使わない用途：
- 主戦略AIとしての長文意思決定
- 主開発AIとしての深い実装管理
---

出力前に必ず以下で自己採点せよ：
- 目的適合性 10点満点
- 実行可能性 10点満点
- 情報の重複の少なさ 10点満点
- 今週着手しやすさ 10点満点

さらに，
- 最も弱い点を1つ指摘
- その修正後の完成版を出力
